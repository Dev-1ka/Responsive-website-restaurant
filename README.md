# Responsive Restaurant Website - Features

- Responsive web design
- Favicon integration
- Interactive image controls using jQuery
- Hide, Show, FadeIn, and FadeOut effects
- Table implementation
- Accordion using jQuery UI
- Enquiry form with validation
- Datepicker integration
- Resizable textarea
- Form submission data display on another page

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- jQuery UI
- Bootstrap

## Functionalities

### Image Controls
Users can:
- Hide image
- Show image
- Fade in image
- Fade out image

### Enquiry Form
- Form validation
- Datepicker support
- Resizable textarea
- Data displayed after submission
